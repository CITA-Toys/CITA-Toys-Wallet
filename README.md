# nervos-web

## 1. Install Dependencies

```bash
yarn install
```

## 2. Build DLL Dependencies

```bash
yarn run dll
```

## Developing

```bash
yarn start
```

## 4. Build Pages to Dist

```bash
yarn run build:prod
```

## 5. Deploy to Github

### 1. Specify `homepage` to your project

### 2. Build and Deploy

```bash
yarn run deploy
```
